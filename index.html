Manual do Inventor 📖

Deslize para o lado para entender a língua das máquinas antes de montar o código.

1. A Biblioteca

#include <stdio.h>

É o manual de instruções. Ensina o computador a ler e escrever na tela.

2. A Chave

int main() { ... }

O coração do programa. É aqui que você dá a partida no motor.

3. A Gaveta

char grupo[50];

A memória. Criamos uma "caixa" invisível para guardar o nome do seu grupo.

4. O Megafone

printf("...");

Como o computador fala com você. Tudo entre aspas aparece na tela.

5. O Ouvido

scanf("%s", grupo);

O computador prestando atenção no teclado e guardando na "gaveta".

6. A Chegada

return 0;

A bandeira quadriculada! Avisa que a missão foi cumprida sem erros.

Blocos de Código (Arraste para ordenar)

  • int main() { ... } 🔑 Onde tudo começa
  • return 0; 🏁 Missão cumprida
  • #include <stdio.h> 📚 Manual de Instruções
  • printf("Qual o nome do grupo?"); 📢 O computador fala
  • scanf("%s", grupo); 👂 O computador ouve
  • char grupo[50]; 📦 Espaço para guardar nomes
